# Ovation Python API


Ovation Python API


## Requirements

* Java 1.6+
* CPython 2.7+ or 3.3+
* NumPy

## Building

To build the python wrapper at the command line:

	mvn compile

Build requires:
* Java 1.6 or later
* Maven 3 or later
* CPython 2.7 or later
